PerpusLing For Admin

  1. Admin membuka website perpusling.com
  2. Admin memasukan username dan password lalu klik lanjutkat/simpan/submit
  3. Admin akan dibawa ke halaman beranda setelah berhasil masuk, dan bisa melakukan aktifitas lainnya
  4. klik transaksi jika ingin melakukan pelaporan transaksi hingga pelayanan pembelian online atau hal lain yg berhubungan dengan transaksi
  5. klik profil untuk melihat informasi lengkap PerpusLing
